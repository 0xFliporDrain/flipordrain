# controls

quick reference for keyboard + mouse + touch on the flip page.

## keyboard

| key      | what it does                              |
|----------|--------------------------------------------|
| `space`  | flip the coin (only when amount is set)   |
| `1`-`5`  | preset amounts: 0.1 / 0.5 / 1 / 2 / 5 SOL |
| `c`      | toggle compact view (hides leaderboard)   |
| `r`      | reconnect ws if the live feed stalls      |
| `?`      | show this list as an in-app overlay       |
| `esc`    | close any toast / modal                    |

## mouse

- click coin: flip
- click leaderboard row: copy wallet to clipboard
- shift+click amt: doubles current amount
- right-click coin: switch sides preview (no tx)

## touch (mobile)

- tap coin: flip
- swipe left on amt input: clear
- pull-to-refresh leaderboard
- long-press a feed row: copy tx sig
