# FlipOrDrain

![Solana](https://img.shields.io/badge/Solana-black?logo=solana) ![Anchor](https://img.shields.io/badge/Anchor-blueviolet) ![Status](https://img.shields.io/badge/status-building-yellow) ![License](https://img.shields.io/badge/license-MIT-green)

On-chain coin flip. 50/50. Win 1.9x or lose it all. Double-or-nothing if you dare.

## Stack

`Anchor` · `Vite + React` · `Switchboard VRF` · `WebSocket live feed`

## Run

```bash
anchor build && anchor deploy
cd app && npm i && npm run dev
```
